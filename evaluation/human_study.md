# Human Readability Study Template

A human study is optional until automatic cross-OCR results pass G7.

## Questions

Use separate questions for:

1. General visual preference at matched total bpp.
2. Text readability.
3. Exact transcription choice for selected words/numeric fields.

Do not combine readability and aesthetics into one ambiguous score.

## Design

- randomized, blinded pairwise comparison
- left/right counterbalancing
- original image available only in a separately defined fidelity task
- preregistered sample selection stratified by domain and text size
- include failure cases, not only proposed-method wins
- screen calibration/instruction checks where feasible

## Participants and ethics

Obtain institutional approval or exemption as required. Avoid exposing private document content; use licensed public data or synthetic examples. Record inclusion/exclusion rules and compensation.

## Analysis

Use participant- and image-aware mixed-effects logistic regression for pairwise preference where sample size permits. Report raw preference, confidence interval, agreement and attention-check failures.
