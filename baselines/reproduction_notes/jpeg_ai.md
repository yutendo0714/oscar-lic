# JPEG AI Reference Anchor

JPEG AI is a standards/reference-software anchor, not a Python baseline. Record:

- exact reference software tag/commit
- profile and operating configuration
- normative/verification model status
- build compiler and flags
- encoder/decoder command
- conformance status
- complete codestream/container bytes
- color conversion and bit depth
- patent/standard licensing review separate from repository code license

Do not compare a selected JPEG AI profile against LIC results without documenting whether the task is human reconstruction, compressed-domain machine use, ROI/progressive coding, or a specific conformance point.
